What is p5.js

P5.js is a creative coding library built with javaScript. It is not, by any means, the only way to do creative coding, but it has a few notable benefits. First, it is open source and runs in any web browser, meaning it is free and easy to use on the internet. That makes it well suited for embedding in websites and supporting user interaction. Second, p5.js is popular, so you can easily find examples and tutorials for most problems. Third, since it is based on processing (another popular creative coding library built with Java), you will learn patterns that can be applied to both.
